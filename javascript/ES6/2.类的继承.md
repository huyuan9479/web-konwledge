### 一、继承
继承：子类可以继承父类的一些属性和方法。

语法：
```javascript
    class Father { //父类
        // ...
    }
    class Son extends Father { //子类继承父类
        // ...
    }
```

例如：
```javascript
class Father {
    constructor() {
        
    }
    money() {
        console.log(1000000)
    }
}

class Son extends Father {
    
}

var son = new Son()
son.money() // 1000000
```
